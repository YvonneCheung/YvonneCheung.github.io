VSCode 常用插件及setting配置

Spt 27, 2020 vscode

记录vscode每次需安装的常用扩展插件，以及自动格式代码的配置。

# 常用插件

1. Auto Rename Tag

自动完成另一侧标签的同步修改

2. Auto Close Tag

自动闭合HTML/XML标签

3. Vetur

Vue 的语法高亮、智能感知、Emmet 等

4. background

修改vscode背景图

5. Bracket Pair Colorizer 2

让括号拥有独立的颜色，易于区分

6. Regex Previewer

将正则表达式模式应用在任何打开的文件上，并高亮所有的匹配项

7. Image Preview

鼠标悬浮在链接或者装订线(gutter)左边可以预览到图片

8. Settings Sync

Visual Studio代码设置同步，需要github账号辅助的

9. IntelliSense for CSS class names

智能提示 css 的 class 名

# setting配置

{
  "workbench.colorTheme": "Ayu Mirage",
  "workbench.iconTheme": "vscode-great-icons",
  "auto-rename-tag.activationOnLanguage": [
    "/dist",
    "*"
  ],
  //background 的相关配置
  "update.enableWindowsBackgroundUpdates": true,
  "background.customImages": [
    "E:/pic/14.jpg" //图片地址路径
  ],
  "background.style": {
    "content": "''",
    "pointer-events": "none",
    "position": "absolute", //图片位置
    "width": "100%",
    "height": "100%",
    "z-index": "9999",
    "background.repeat": "no-repeat",
    "background-size": "cover",
    "background-postion": "100%,100%", //图片大小
    "opacity": 0.2 //透明度
  },
  "background.useFront": true,
  "background.useDefault": false,
  //搜索忽略
  "search.exclude": {
    "/node_modules": true,
    "/bower_components": true,
    "/*.code-search": true,
    "/dist": true
  },
  // vscode默认启用了根据文件类型自动设置tabsize的选项
  "editor.detectIndentation": false,
  // 重新设定tabsize
  "editor.tabSize": 2,
  // #每次保存的时候自动格式化
  "editor.formatOnSave": true,
  // #每次保存的时候将代码按eslint格式进行修复
  // "eslint.autoFixOnSave": true,
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  },
  // 添加vue支持
  "eslint.validate": [
    "javascript",
    "html",
    "vue"
  ],
  // //  #去掉代码结尾的分号
  // "prettier.semi": false,
  // //  #使用带引号替代双引号
  // "prettier.singleQuote": true,
  //  #让函数(名)和后面的括号之间加个空格
  "javascript.format.insertSpaceBeforeFunctionParenthesis": true,
  // #这个按用户自身习惯选择
  "vetur.format.defaultFormatter.html": "js-beautify-html",
  // #让vue中的js按编辑器自带的ts格式进行格式化
  "vetur.format.defaultFormatter.js": "vscode-typescript",
  "vetur.format.defaultFormatterOptions": {
    "js-beautify-html": {
      "wrap_attributes": "force-aligned"
      // #vue组件中html代码格式化样式
    }
  },
  // 文件末尾加新行
  "files.insertFinalNewline": true,
  "terminal.integrated.shell.windows": "C:\\Windows\\System32\\cmd.exe",
  "window.zoomLevel": 0,
  "editor.foldingStrategy": "indentation"
}
