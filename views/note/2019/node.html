Node.js全栈入门

Spt 01, 2019 node.js

监控文件的变化，自动执行相应的命令

# 环境准备

# nodemon

监控文件的变化，自动执行相应的命令

  • 配置 nodemon
  1. npm install nodemon -D(在 dev 环境下安装)
  2. 修改 package.json 中的启动命令
  3. 通过增加 nodemon.json 配置指定特殊 watch 的文件
  4. 增加"DEBUG=*"查看日志

# nrm

管理 npm 源的工具

  • 安装 nrm
    npm install nrm -g
  • 查看源
    nrm ls
  • 切换源
    nrm use taobao

# nvm

管理 nodejs 版本

  • 查看远程端 node 现有版本
    nvm ls-remote --lts
  • 切换 node 和 npm 版本(只在当前命令窗口有效)
    nvm use v10.16.0
  • 设置 default 版本
    nvm alias default v10.16.0
